# Golf Tracker

Mobile golf scorecard web app.

## Features

- 2, 3 or 4 players
- Gross and Stableford scoring modes
- Handicap (0-54) with Stroke Index distribution
- 10+ Russian golf courses with auto Par and SI
- Hole-in-One highlight on Par 3
- Final scorecard with share and save
- PWA - works offline
